# 블로그 추가 기능 전체 정리 (통합본)

현재 구현됨: **게시글 CRUD/검색, 댓글, 회원가입/로그인/로그아웃, 공개/비공개, Bootstrap UI(다크모드), Django Admin**

---

## A. UX/콘텐츠 탐색 개선 (체감 큼 · 쉬움~보통)

### 1) 카테고리 / 태그
- 글 분류 & 탐색성 향상
- 카테고리별/태그별 글 목록 페이지

### 2) 페이지네이션 + 정렬
- 페이지네이션: 글이 많아질수록 필수
- 정렬: 최신/조회수/좋아요 등

### 3) 조회수(views) + 인기글 위젯
- 조회수 증가 로직 + 중복 방지(세션/쿠키/IP 등)
- 사이드바/메인에 “인기글 Top N”

### 4) 임시저장(Draft) + 예약발행
- Draft 저장 / Published 전환
- 예약 시간(`published_at`) 도달 시 공개

### 5) 이미지 업로드 + 본문 이미지 삽입
- 글 작성 시 이미지 업로드 및 본문에 삽입
- media 설정(MEDIA_URL/MEDIA_ROOT)

### 6) 코드블록 하이라이트
- 개발 블로그 완성도 상승
- 코드블록 스타일/복사 버튼(옵션)

---

## B. 블로그 “완성도/노출” 기능 (보통)

### 7) SEO 기본 세팅
- slug(주소), meta description
- Open Graph(공유 미리보기), canonical

### 8) RSS 피드 + sitemap.xml
- 구독/배포/검색 크롤링에 유리
- Django feed / sitemap 활용

### 9) 관련 글 추천
- 같은 태그/카테고리 기반 추천
- 체류시간/회전율 상승

### 10) 댓글 운영 기능 (moderation) + 신고
- 숨김/승인/차단 처리
- 신고 누적 기반 자동 숨김(옵션)

### 11) 프로필/소개 페이지
- 포트폴리오 감성 강화
- 자기소개/기술스택/외부 링크/GitHub 등

---

## C. 운영/보안/품질 강화 (보통~어려움)

### 12) 비밀번호 재설정 이메일 + 이메일 인증
- 신뢰도/편의성 상승
- 개발(콘솔) → 운영(SMTP) 전환

### 13) Rate Limit / 스팸 방지
- 로그인/댓글 도배 방지
- IP/계정 기반 제한(+ 캡차 옵션)

### 14) 백업/복구 커맨드 + 관리자 대시보드
- 운영 관점에서 강력한 포인트
- 백업 생성/다운로드/복구 흐름

---

5) **SEO + RSS + sitemap**  
6) **댓글 moderation/신고**  
7) **이메일 인증/비번 재설정**  
8) **Rate limit**  
9) **백업/복구 + 운영 대시보드**
